Python Cookbook 3rd Edition Notes
================================================

本文档是对 **《Python Cookbook, 3rd Edition》** 的阅读笔记, 在熊能翻译的 `中文版 <http://python3-cookbook.readthedocs.io>`__ 的基础上修改而成. 源文档托管于 github, 项目主页为 https://github.com/CarlKing5019/python3-cookbook-notes


《Python Cookbook》英文版有 706 页, 中文版有 619 页, 出版于 2013 年 5 月, 包含 15 个方面，共计 260 个例子, 介绍了 Python 应用在各个领域中的一些使用技巧和方法, 覆盖了 Python 应用中的很多常见问题, 并提出了通用的解决方案, 适合有一定 Python 基础的用户阅读.
